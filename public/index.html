📖 Instructions d'Installation

  1. Téléchargez le package auto-extractible (recommandé)
  2. Décompressez le fichier .zip sur votre bureau
  3. Double-cliquez sur INSTALL.command
  4. Suivez les instructions à l'écran
  5. Lancez Sentinel Agent depuis Applications ou Desktop
  6. Enrollez l'agent avec votre token depuis le dashboard

🔧 Installation Alternative

Pour une installation manuelle ou avancée :

  1. Ouvrez Terminal
  2. Naviguez vers le dossier téléchargé
  3. Exécutez : ./install-client.sh
  4. Suivez les instructions interactives

⚠️ Note de Sécurité

Ce logiciel n'est pas signé par Apple, ce qui déclenche Gatekeeper. C'est normal pour les logiciels d'entreprise. Le package inclut des vérifications de sécurité et d'intégrité pour garantir la fiabilité de l'installation.
