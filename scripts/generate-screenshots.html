📸 Générateur de Screenshots App Store

Capturez vos screenshots aux dimensions exactes requises

📋 Instructions de Capture

  1. Sélectionnez le device (iPhone 6.7", 6.5" ou iPad 12.9")
  2. Sélectionnez la page à capturer (5 pages au total)
  3. Attendez le chargement complet de la page
  4. Utilisez Cmd+Shift+4 puis Espace pour capturer la fenêtre iframe
  5. Ou clic droit sur l'iframe > "Ouvrir dans un nouvel onglet" puis Cmd+Shift+3
  6. Répétez pour les 3 devices × 5 pages = 15 screenshots
  7. Nommez les fichiers :
    • iphone-6.7/1-dashboard.png
    • iphone-6.7/2-assets.png
    • etc...

⚠️ Alternative Automatique : Si l'app web n'est pas démarrée, lancez d'abord :

npm run dev

Puis ouvrez cette page et l'iframe chargera automatiquement l'application.
