## Challenges Faced

### 1. Zod v4 API Changes
- **Story:** 1.2 (Humanized Error Messages)
- **Issue:** Zod v4 changed error handling API
- **Resolution:** Created schema factory functions instead of global error map
- **Learning:** Check library changelogs before implementation

### 2. Async Testing with Fake Timers
- **Stories:** 1.4, 1.5
- **Issue:** Vitest fake timers complexity with debounced functions
- **Resolution:** Used synchronous `safeParse` instead of `parseAsync`
- **Learning:** Prefer synchronous patterns when possible for easier testing

### 3. Double-Click Prevention
- **Story:** 1.6
- **Issue:** React state updates too slow for synchronous guard
- **Resolution:** Used `useRef` for immediate synchronous check
- **Learning:** `useRef` for synchronous guards, `useState` for UI updates
